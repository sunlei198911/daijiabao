# 登录接口

## url

```
POST: /api/login
```

## 入参

`username`  用户名
`password`  密码

## 出参

### 登录成功

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": true,
        "userId": "asd8664sdnhjsfbblljssskuppqzk"
    }
}
```

`data.isSuccess` 是否登录成功
`data.userId` 登陆后的用户 ID，为将来扩展预留，可暂时不实现

### 登录失败

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": false
    }
}
```
