# 信息上传保存

> 用于信息上传模块。

## url

```
POST: /api/accountBills/saveInfo
```

    "id": "",

                {
                    "kay1": "www",

## 出参

保存成功

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": true
    }
}
```

保存失败

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": false
    }
}
```
