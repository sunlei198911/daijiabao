# 退出登录接口

## url

```
POST: /api/logout
```

## URL入参

`userID`  用户 ID

## 出参

### 退出登录成功

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": true
    }
}
```

`data.isSuccess` 退出登录是否成功

### 退出登录失败

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": false
    }
}
```
