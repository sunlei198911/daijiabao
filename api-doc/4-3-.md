# 强制报税

> 用于信息上传模块。

## url

```
POST: /api/focus-bills/:companyId
```

companyId，公司 id

## 入参

无

## 出参

报税成功

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": true
    }
}
```

报税失败

```json
{
    "code": 0,
    "message": "",
    "data": {
        "isSuccess": false
    }
}
```
